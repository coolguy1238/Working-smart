Help me write a Python function that finds duplicate entries in a list and returns each duplicate alongside how many times it appears.
V
Here's a clean solution using Python's Counter from the collections module:

from collections import Counter

def find_duplicates(lst):
    counts = Counter(lst)
    return {item: count for item, count in counts.items() if count > 1}

This returns a dictionary of only the duplicates and their counts. Want me to add type hints, or handle edge cases like nested lists?
